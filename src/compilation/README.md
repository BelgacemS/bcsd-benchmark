## Compilation

Ce dossier contiendra le pipeline de compilation des codes sources en binaires

### Objectif

Compiler les implémentations collectées (C, C++, Rust, Go) avec différentes configurations :
- Compilateurs : gcc, clang, rustc, go
- Niveaux d'optimisation : -O0, -O1, -O2, -O3
- Architectures : x86_64, ARM (si possible)
