## Évaluation

Ce dossier contiendra les scripts d'évaluation et de benchmark pour la détection de similarité de code binaire

### Objectif

Évaluer les approches état de l'art sur le benchmark construit :
- Baselines (features manuelles, graph-based methods)
- Approches LLM (Trex, PalmTree, SAFE)
- Architectures Transformer (k-trans)
- Méthodes d'embedding (Asm2Vec)
